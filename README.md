# Mini Guia de Comandos Essenciais do Linux com NotebookLM

## 📚 Contexto

Este projeto foi criado como parte de um desafio da DIO para explorar o uso da Inteligência Artificial como ferramenta de aprendizado.

Utilizando o NotebookLM, foram analisados materiais abertos sobre Linux para criar um mini guia com comandos essenciais para iniciantes.

O objetivo é demonstrar como a IA pode ajudar na organização de conhecimento técnico.

---

## 🎯 Objetivos de Estudo

- Entender comandos básicos do Linux  
- Criar um material de revisão simples  
- Testar engenharia de prompts com IA  
- Organizar conhecimento em formato de guia  

---

## 📖 Mini Guia de Estudo

### ls

Lista arquivos e diretórios.

Exemplos:

ls
ls -l
ls -a

---

### cd

Navegar entre diretórios.
cd /home
cd ..
cd pasta

---

### pwd

Mostra o diretório atual.
pwd

---

### cp

Copiar arquivos.
cp arquivo.txt backup.txt

---

### mv

Mover ou renomear arquivos.
mv arquivo.txt pasta/

---

### rm

Remover arquivos.
rm arquivo.txt

---

## 📘 Glossário

**Shell**  
Interface de comando do sistema.

**Terminal**  
Aplicação usada para interagir com o shell.

**Diretório**  
Pasta dentro do sistema de arquivos.

**Root**  
Usuário administrador do sistema.
